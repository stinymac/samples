###ln 指令
    软链接也成为符号链接，类似于windows里的快捷方式，
    主要存放了链接其他文件的路径
    
    基本语法
    ln -s [原文件或目录] [软链接名] （功能描述：给原文件创建一个软链接）
    
    删除软连接
    
    rm -rf [软链接名]
    
###history指令

    查看已经执行过历史命令,也可以执行历史指令
    
    基本语法
    history （功能描述：查看已经执行过历史命令）   
    
    history 10
    !10  执行编号为10的历史命令
