###date指令

####显示当前日期

    基本语法
    1) date （功能描述：显示当前时间）
    2) date +%Y （功能描述：显示当前年份）
    3) date +%m （功能描述：显示当前月份）
    4) date +%d （功能描述：显示当前是哪一天）
    5) date "+%Y-%m-%d %H:%M:%S"（功能描述：显示年月日时分秒）
    
####设置日期
    
    基本语法
    date -s 字符串时间
    
###cal指令
    
    基本语法
    cal [选项] （功能描述：不加选项，显示本月日历）
